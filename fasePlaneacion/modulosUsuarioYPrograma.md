## Módulo Usuario
El módulo de usuario va a permitir a Diana administrar la base de datos, por lo que cuenta con atributos tales como:
_usuario, contraseña, identificación y rol._

Adicionalmente, el módulo será capaz de modificar sus propios atributos a través de diversos métodos como:
_iniciarSesion() y gestionarRolesUsuario()._

## Módulo Programa
El módulo programa será capaz de gestionar capacitaciones dentro de sí, razón por la cual cuenta con atributos como: _nombreDePrograma_ y _faseDelPHVA_ para cada capacitación. 

Para manejar estas, también cuenta con métodos como:
_agregarCapacitacionAPrograma()_ y _modificarFasePHVACapacitacion()._
